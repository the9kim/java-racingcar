- 자동차 경주 기능 목록
    - 메인 로직
        - 자동차 생성
            - [X] 자동차 이름을 입력 받는다.
            - [X] 입력값을 검증한다.
            - [X] 자동차를 생성한다.
        - 자동차 경주
            - [X] 시도 횟수를 입력 받는다.
            - [X] 시도 횟수를 검증한다.
            - [X] 자동차를 이동한다.
                - [X] 랜덤 숫자를 생성한다.
                - [X] 자동차를 이동시킨다.
            - [X] 차수별 실행 결과를 출력한다
        - 결과 출력
            - [X] 최종 우승자를 출력한다.

    - 예외처리
        - 아래 경우 예외처리 후 다시 입력받는다.
            - 자동차 이름입력값
                - (포맷) 입력값 포맷에 맞지 않는 경우
                    - [X] 공백
                    - [X] 자동차 이름이 5자를 초과
                    - [X] 자동차 대수가 2대 미만일 경우
                    - [X] 자동차 간 이름이 중복일 경우
            - 시도 횟수
                - [X] 공백
                - [X] 정수x (1이상, Integer 범위)
